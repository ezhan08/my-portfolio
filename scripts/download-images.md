# How to Download Images from LinkedIn

## Method 1: Right-Click Download

1. Navigate to your LinkedIn profile or posts
2. Right-click on the image
3. Select "Save image as..." or "Download image"
4. Save to: `public/images/` folder
5. Name them descriptively:
   - `profile-picture.jpeg` (for profile photo)
   - `post-project-1.jpg` (for project images)
   - `post-achievement-1.jpg` (for achievement posts)

Use a browser extension like:

- "Download All Images" for Firefox

## Recommended Image Names

- Profile picture: `profile-picture.jpeg`
- Post images: `post-[description].jpg`
- Project screenshots: `project-[name].jpg`
- Certificates: `cert-[name].jpg`

## Image Requirements

- **Profile Picture**: At least 400x400px, square or portrait
- **Post Images**: Any size, will be optimized
- **Format**: JPG, PNG, or WebP
